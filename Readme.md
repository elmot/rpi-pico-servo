DIY servo
====
Components
----

* Raspberry Pi Pico
* AS5600
* DRV8833

Pinout
----

| Func   | Rpi GPIO | Pin | Part   |
|--------|----------|-----|--------|
| SDA    | GP26     | 31  | AS5600 |
| SCL    | GP27     | 32  | AS5600 |
| PWM1   | GP16     | 21  | DRV    |
| PWM2   | GP17     | 22  | DRV    |
| PWM IN | GP21     | 27  | Input  |
